
# 🏨 OceanView Hotel Management System

A web-based **Hotel Booking & Reservation System** built with **Java EE, JSP, Servlets, and MySQL** — managed through NetBeans and deployed on **Apache Tomcat**.


---

## 🛠️ Built With
- **Frontend:** HTML, CSS, JavaScript, JSP  
- **Backend:** Java Servlets  
- **Database:** MySQL  
- **Server:** Apache Tomcat / TomEE  
- **IDE:** NetBeans  

---

## ✅ Prerequisites
Before you begin, make sure you have installed:

- ☕ Java JDK 19  
- 🖥️ NetBeans IDE  
- 🐱 Apache Tomcat or TomEE  
- 🐬 MySQL Server  

---

## 📦 Required JAR Files
Add these to `WEB-INF/lib/`:

- `jakarta.jakartaee-api-10.0.0.jar`  
- `mysql-connector-j-9.6.0.jar`
